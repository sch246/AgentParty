### Phase 4: 工具参数系统 ✅
**目标：** 支持 `do py timeout=60` 等带参数的工具调用

**成果：**
- ✅ 参数解析系统（`key=value` 语法）
- ✅ Python timeout 支持（线程控制）
- ✅ Shell timeout + cwd 支持
- ✅ 自动类型转换和验证
- ✅ 向后兼容（无参数时保持原行为）
- ✅ **109 个测试全部通过**（新增 19 个）

**支持的参数：**
- `timeout=<秒>`: Python 和 Shell 通用
- `cwd=<路径>`: Shell 专用

**语法示例：**
```
do py timeout=60
    time.sleep(30)
end

do sh cwd=/tmp timeout=5
    pwd && ls
end
```
