## 🛠️ 使用指南

### 快速开始

**File 模式：**
```bash
python file_talk.py
# 编辑 .md 文件，以 \n\n 结尾触发
```

**CLI 模式：**
```bash
python cli_talk.py
# 交互式对话，支持工具调用
```

**运行测试：**
```bash
python -m unittest discover tests -v
# 或使用快捷脚本
./run_tests.sh      # Linux/Mac
run_tests.bat       # Windows
```

### 配置文件 (.config.json)
```json
{
  "api_key": "your-key",
  "base_url": "https://api.openai.com/v1",
  "model": "gpt-4",
  "debug": true,
  "max_retries": 3,
  "max_tool_rounds": 10
}
```
